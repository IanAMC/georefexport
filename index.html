QGIS SBM Tools

  • GeorefExport - Exports the map as a georeferenced raster graphics file.
    • Input:
      • Project CRS: Project Coordinate Reference System
      • DPI: Resolution of the output image
      • WIDTH [pixel]: Pixel width of the output image
      • Image format: raster graphics format file supported by the local gdal installation
        (PDF, GTiff, JPEG, PNG, JPEG2000, ECW, JP2ECW, JP2MrSID, JP2OpenJPEG)
      • Worldfile: Force the generation of an associated ESRI world file
    • Output:
      • Output georeferenced raster graphics file
    • References:
      • www.gdal.org (gdal_translate).
    • Author:
      • Ruggero Valentinotti http://www.bacinimontani.provincia.tn.it
